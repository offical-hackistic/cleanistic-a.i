Cleanistic AI Widget Integration Demo

This page demonstrates how to embed the Cleanistic AI estimator widget into any website.

Basic Integration

Add these two lines to your HTML:

<script src="https://cdn.cleanistic.ai/widget.js"></script> <cleanistic-estimator company-id="demo" primary-color="#3b82f6"></cleanistic-estimator>

Live Widget Demo:

Customization Options

<cleanistic-estimator company-id="your-company-id" api-key="your-api-key" primary-color="#10b981" secondary-color="#3b82f6" company-name="ABC Cleaning Co" enable-house-washing="true" enable-roof-cleaning="true" enable-gutter-cleaning="false" require-address="true"> </cleanistic-estimator>

Event Handling

<script> // Listen for estimate completion document.addEventListener('cleanistic:estimate-complete', (event) => { const { analysisId, totalEstimate } = event.detail; console.log('Estimate completed:', totalEstimate); // Redirect to contact form or show success message window.location.href = `/contact?estimate=${analysisId}`; }); // Listen for errors document.addEventListener('cleanistic:error', (event) => { console.error('Estimator error:', event.detail.error); }); </script>

WordPress Integration

For WordPress sites, use our shortcode:

[cleanistic-estimator company_id="your-company-id" primary_color="#3b82f6"]

React Integration

import { CleanisticEstimator } from '@cleanistic/react-estimator'; function App() { return ( <CleanisticEstimator companyId="your-company-id" config={{ branding: { primaryColor: "#3b82f6", companyName: "Your Cleaning Co" } }} onEstimateComplete={(result) => { console.log('Estimate:', result); }} /> ); }
